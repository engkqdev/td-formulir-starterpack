## 📝 Starterpack System With Textdraw

Ini adalah system menclaim starterpack dari pemerintahan, untuk kota **Village State Roleplay**.

### **Workflow & Logic**
1. **Command Trigger**: Warga baru bisa memunculkan Textdraw dengan mengetik perintah `/starterpack`.
2. **Identification**: System akan memunculkan Textdraw Formulir yang mewajibkan pemain mengisi Nama Lengkap karakter.
3. **Automated Grant**: Setelah data divalidasi, System secara otomatis memberikan modal uang sebagai bantuan dari pemerintah.
4. **One-Time Validation**: Fitur ini dilengkapi dengan sistem pengecekan database untuk memastikan setiap warga hanya dapat mengklaim bantuan satu kali.
